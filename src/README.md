# ⌨️ Coding Speed Typin

Uno Speed Type Test pensato per programmatori: invece di
parole a caso, scrivi vere righe di codice. Ispirato a MonkeyType, ma per chi
passa più tempo su una tastiera a scrivere `const` che frasi di senso compiuto.

![Visitor Count](https://visitor-badge.laobi.icu/badge?page_id=thatsomx.coding-speed-typer)
![License: MIT](https://img.shields.io/badge/license-MIT-673AB7)

## ✨ Funzionalità

- Scegli il linguaggio: JavaScript, TypeScript, Python, Java o C++
- Countdown di 3 e 30-120 secondi di typing a tempo, con feedback visivo in tempo reale
  (carattere corretto, errore, non ancora scritto)
- Resoconto finale: WPM, precisione, errori, caratteri corretti, durata
- Transizioni fluide fra le schermate (View Transition API nativa)
- Estetica ispirata a un editor di codice (VS Code), palette scura e
  dettagli d'interazione (puntini di sfondo reattivi al mouse, bottoni con
  effetto luminoso rettivo)

## 🛠️ Stack tecnico

- React + TypeScript
- Vite
- CSS

## 💻 Sviluppo in locale

```bash
npm install
npm run dev
```

Apri `http://localhost:5173` (o la porta indicata nel terminale).

## 📦 Build di produzione

```bash
npm run build
npm run preview   # per testare la build in locale
```
